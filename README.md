## ✨ Fonctionnalités

### Version Gratuite (sans inscription)
- 4 templates CV professionnels (Moderne, Minimaliste, Créatif, Corporate)
- Aperçu en temps réel
- Export PDF haute qualité A4
- Export Word / RTF / TXT
- Générateur de lettre de motivation (3 variations)
- Score CV sur 100 avec recommandations
- Match avec offre d'emploi (analyse de mots-clés)
- Traduction automatique FR → EN
- Générateur de posts LinkedIn (6 types)
- Simulation d'entretien (6 modes, score /100, rapport)
- Import d'ancien CV (PDF, TXT, Word)
- Recadrage de photo intégré (Canvas)
- Dark / Light mode
- Historique des versions (5 slots)
- Multi-profils CV (jusqu'à 10)
- Sauvegarde automatique (LocalStorage)

### Version Pro — 3 500 FCFA (paiement unique)
- 6 templates supplémentaires : Executive, Timeline, Élégant, Colorblock, Compact, Gradient Pro
- Mode Présentation plein écran
- Portfolio web exportable (HTML)
- QR Code de contact
- Toutes les futures fonctionnalités Pro

## 🛠 Technologies
- HTML5 / CSS3 / JavaScript vanilla (aucune dépendance)
- Canvas API (recadrage photo circulaire)
- Web Crypto / SHA-256 (codes Pro)
- LocalStorage (sauvegarde automatique)

- QRCode.js (génération QR Code)
