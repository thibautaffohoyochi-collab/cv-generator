# CV Generator Pro

**Créateur de CV professionnel en ligne** — par Thibaut A. Hoyochi

## 🚀 Accès direct
👉 [Ouvrir le CV Generator](https://thibautaffohoyochi-collab.github.io/cv-generator/)

## ✨ Fonctionnalités
- 4 templates CV professionnels (Moderne, Minimaliste, Créatif, Corporate)
- Aperçu en temps réel
- Export PDF haute qualité
- Export Word / RTF / TXT
- Générateur de lettre de motivation (3 variations)
- Score CV sur 100 avec recommandations
- Match avec offre d'emploi (analyse de mots-clés)
- Traduction automatique FR → EN
- Générateur de posts LinkedIn (6 types)
- Mode présentation plein écran
- Portfolio web exportable
- QR Code de contact
- Historique des versions
- Import d'ancien CV (PDF, TXT, Word)
- Recadrage de photo intégré
- Dark / Light mode
- 100% gratuit, sans inscription

## 🛠 Technologies
- HTML5 / CSS3 / JavaScript vanilla
- Canvas API (recadrage photo)
- LocalStorage (sauvegarde automatique)
- Font Awesome 6
- Google Fonts (Poppins, Inter)
